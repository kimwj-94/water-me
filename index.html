2단계: 네이버 캘린더 & 와이프 공유 설정 가이드

1
구글 캘린더 공유하기 (와이프 계정 추가)

PC로 구글 캘린더에 접속한 후, 새로 만들어진 '물 줘! 식물 관리' 캘린더 설정에서 [특정 사용자와 공유]를 클릭하고 와이프분의 구글 이메일을 추가해 줍니다.

2
본인의 네이버 캘린더와 연동

네이버 캘린더 웹 설정 > [내 캘린더] > [외부 캘린더 불러오기]에서 '구글 캘린더 연동'을 선택해 구글 계정을 로그인합니다. '물 줘!' 일정이 네이버 캘린더에 표시됩니다.

3
와이프의 네이버 캘린더 연동

와이프분도 마찬가지로 네이버 캘린더 설정에서 자신의 구글 계정(캘린더를 공유받은 계정)을 연동합니다. 이제 부부 모두 평소 쓰던 네이버 캘린더에서 실시간 물주기 일정을 한눈에 공유할 수 있습니다! 💑

회사 PC의 식물 관리 데이터를 집 노트북이나 다른 환경으로 아주 간단하게 백업하고 즉시 이동할 수 있습니다.

cloud_download

1. 내보내기 (Export)

회사 PC 등 현재 기기에서 가꾼 화분들의 모든 데이터를 파일 하나로 안전하게 다운로드합니다.

  • 저장 대상: 모든 식물 목록, 마지막 물준 날짜, 물주기 주기, 전용 별칭
  • 파일 형식: water_me_backup.json (초경량 텍스트 파일)
cloud_upload

2. 가져오기 (Import)

    🔎 식물 정보가 더 필요하신가요?

    입력하신 식물명을 기반으로 인터넷에서 직접 물주기 팁을 찾아보실 수 있습니다.

    🔑 구글 API Credentials 무료 발급 가이드

    1. 1. 구글 클라우드 콘솔 접속

      Google Cloud Console에 본인의 구글 계정으로 로그인한 뒤, 상단에서 [새 프로젝트 생성]을 진행합니다.

    2. 2. API 및 서비스 활성화

      왼쪽 메뉴의 [API 및 서비스] > [라이브러리]로 이동하여 "Google Calendar API"를 검색하고 [사용] 버튼을 누릅니다.

    3. 3. OAuth 동의 화면 구성

      [API 및 서비스] > [OAuth 동의 화면] 탭으로 이동합니다.
      • User Type: 외부(External)로 지정
      • 앱 정보 입력 후 다음단계로 이동
      • 테스트 사용자: 자신의 구글 이메일 및 와이프분의 구글 이메일을 반드시 추가해 줍니다. (테스트 모드이므로 필수)

    4. 4. API Key 생성

      [사용자 인증 정보] 탭으로 이동하여 [+ 사용자 인증 정보 만들기] > [API 키]를 클릭하여 키를 생성하고 안전하게 복사해 둡니다.

    5. 5. OAuth 클라이언트 ID 생성

      [사용자 인증 정보] 탭에서 다시 [+ 사용자 인증 정보 만들기] > [OAuth 클라이언트 ID]를 선택합니다.
      • 애플리케이션 유형: 웹 애플리케이션 (Web Application)
      • 승인된 자바스크립트 원본(Origin): 로컬에서 파일로 실행할 경우 http://localhost 또는 http://127.0.0.1을 입력합니다. (나중에 GitHub Pages 등으로 무료 배포 시 해당 배포 도메인을 추가해 줍니다.)
      • 생성이 완료되면 Client ID가 발급되며 이를 복사합니다.

    6. 6. 프로그램에 입력 후 연동 완료!

      복사한 Client ID와 API Key를 본 프로그램의 설정 탭에 붙여넣고 [설정 저장] 후 [구글 계정 연동]을 누르면 완성됩니다!
