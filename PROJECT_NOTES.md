# 물 줘! 배포 메모

이 문서는 배포본에 포함해도 되는 중립적인 프로젝트 메모입니다.

## 목적

`물 줘!`는 개인 또는 가족이 키우는 화분의 물주기 일정을 관리하고, 구글 캘린더에 알림용 일정을 생성하는 정적 웹앱입니다.

## 주요 설계

- 서버 없이 `index.html`, `style.css`, `app.js`, `plants-db.js`만으로 실행합니다.
- 식물 데이터는 기본적으로 브라우저 `localStorage`에 저장됩니다.
- 구글 캘린더 연동을 설정하면 앱이 식물별 미래 물주기 일정을 캘린더에 생성합니다.
- 앱이 원본 데이터이고, 구글 캘린더는 알림/표시용 사본으로 봅니다.
- 네이버 캘린더 같은 외부 캘린더에는 구글 캘린더를 구독하거나 연동해서 보기용으로 사용할 수 있습니다.

## 일정 생성 방식

- 계절별 물주기 보정을 반영하기 위해 무한 반복 일정 대신 앞으로 6개월치 개별 일정을 생성합니다.
- 앱에서 식물을 수정하거나 `물 줬어요!`를 누르면 기존 미래 일정을 정리한 뒤 다시 생성합니다.
- 캘린더에서 일부 일정을 수동 삭제해도 다음 동기화 때 앱 기준으로 복구됩니다.

## 개인정보

이 저장소에는 개인 Google Client ID, API Key, 캘린더 ID가 포함되어 있지 않습니다. 사용자가 앱 화면에서 입력한 설정은 해당 브라우저의 `localStorage`에만 저장됩니다.
