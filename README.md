# 물 줘!

식물 물주기 일정 관리 앱입니다. 브라우저에서 바로 실행되는 정적 웹앱이며, 구글 캘린더와 연동하면 물주기 알림 일정을 자동으로 생성할 수 있습니다.

## 주요 기능

- 화분 등록, 수정, 삭제
- 마지막 물준 날짜 기준 다음 물줄 날짜 계산
- 대한민국 계절별 물주기 주기 자동 보정
- 기본 식물 도감 52종 제공
- 직접 식물 도감 항목 추가
- 구글 캘린더에 앞으로 6개월치 물주기 일정 생성
- 앱 데이터 기준으로 구글 캘린더 물주기 일정 전체 재생성

## 파일 구성

- `index.html`: 앱 화면
- `style.css`: 디자인
- `app.js`: 앱 동작 로직
- `plants-db.js`: 기본 식물 도감 데이터
- `SETUP.md`: 설치와 구글 캘린더 연동 방법
- `PROJECT_NOTES.md`: 배포용 프로젝트 메모

## 빠른 실행

캘린더 연동 없이 식물 관리만 하려면 `index.html`을 브라우저로 열면 됩니다.

구글 캘린더 연동까지 사용하려면 `file://`로 직접 열지 말고, GitHub Pages 같은 웹 주소 또는 로컬 서버 주소로 실행해야 합니다. 자세한 방법은 [SETUP.md](./SETUP.md)를 참고하세요.

## 데이터와 개인정보

이 저장소에는 개인 API Key, Client ID, 캘린더 ID가 포함되어 있지 않습니다.

사용자가 앱 화면에서 입력한 식물 데이터와 구글 설정은 해당 브라우저의 `localStorage`에 저장됩니다. 공용 PC에서는 사용 후 브라우저 사이트 데이터를 삭제하세요.

## 알림 권장 방식

네이버 캘린더 등 외부 캘린더에 구글 캘린더를 연동하면 보기에는 편하지만 알림까지 안정적으로 전달되지 않을 수 있습니다. 알림은 Google Calendar 앱에서 받는 방식을 권장합니다.

## 캘린더 동기화 기준

앱에 저장된 화분 목록이 원본입니다. 구글 캘린더 연동이나 전체 재생성을 실행하면 기존 물주기 일정은 일괄 정리되고, 현재 앱에 남아 있는 화분 기준으로 새 일정이 생성됩니다.
