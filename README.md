# 💧물 줘!

식물 물주기 일정 관리 앱입니다. 브라우저에서 바로 실행되는 정적 웹앱이며, 구글 캘린더를 중앙 저장소로 사용해 PC와 모바일에서 같은 화분 데이터를 불러옵니다.

## 주요 기능

- 화분 등록, 수정, 삭제
- 마지막 물준 날짜 기준 다음 물줄 날짜 계산
- 대한민국 계절별 물주기 주기 자동 보정
- 기본 식물 도감 300종 제공
- 직접 식물 도감 항목 추가
- 구글 캘린더에 앱 중앙 데이터 저장
- 구글 캘린더에 앞으로 6개월치 물주기 일정 생성
- 중앙 데이터 기준으로 구글 캘린더 물주기 일정 전체 재생성
- 저장/삭제/동기화 작업 직렬화로 중복 실행 방지
- 구글 토큰 만료 시 1회 자동 재시도
- 일정 생성/삭제 배치 처리와 실패 시 순차 처리 fallback

## 파일 구성

- `index.html`: 앱 화면
- `style.css`: 디자인
- `app.js`: 앱 동작 로직
- `plants-db.js`: 기본 식물 도감 300종 데이터
- `SETUP.md`: 설치와 구글 캘린더 연동 방법
- `DISTRIBUTION.md`: 주변 사람에게 배포할 때 참고할 간단 안내문
- `PROJECT_NOTES.md`: 배포용 프로젝트 메모

## 빠른 실행

캘린더 연동 없이 식물 관리만 하려면 `index.html`을 브라우저로 열면 됩니다. 다만 이 경우 데이터는 현재 기기에만 저장됩니다.

PC와 모바일에서 같은 데이터를 쓰려면 구글 캘린더 연동이 필요합니다. `file://`로 직접 열지 말고, GitHub Pages 같은 웹 주소 또는 로컬 서버 주소로 실행해야 합니다. 자세한 방법은 [SETUP.md](./SETUP.md)를 참고하세요.

## 데이터와 개인정보

이 저장소에는 개인 API Key, Client ID, 캘린더 ID가 포함되어 있지 않습니다.

구글 연동 전의 임시 데이터와 구글 설정은 해당 브라우저의 `localStorage`에 저장됩니다. 구글 연동 후 화분 목록과 사용자 추가 도감은 지정한 구글 캘린더의 앱 데이터 이벤트에 저장됩니다. 공용 PC에서는 사용 후 브라우저 사이트 데이터를 삭제하세요.

보안과 안정성을 위해 앱은 새 구글 캘린더를 자동 생성하지 않습니다. 사용자가 직접 만든 식물관리 캘린더 ID를 설정에 저장한 뒤, 해당 캘린더의 앱 데이터와 물주기 알림 일정만 관리합니다.

## 알림 권장 방식

네이버 캘린더 등 외부 캘린더에 구글 캘린더를 연동하면 보기에는 편하지만 알림까지 안정적으로 전달되지 않을 수 있습니다. 알림은 Google Calendar 앱에서 받는 방식을 권장합니다.

## 캘린더 동기화 기준

구글 캘린더 안의 앱 데이터 이벤트가 원본입니다. PC와 모바일은 로그인할 때 이 데이터를 먼저 읽고, 변경사항을 저장한 뒤 물주기 알림 일정을 다시 생성합니다.

알림 일정은 새 버전부터 사용자에게 보이는 설명 안에 숨은 식물 데이터를 넣지 않고, 구글 캘린더의 비공개 확장 속성으로만 식별합니다. 예전 버전에서 만든 일정은 정리할 수 있도록 구버전 검색 호환성만 유지합니다.
